Tutorial on Objects

Objects

In JavaScript, an object is a collection of related data or functionality. An object usually consists of several variables and functions often called properties and methods

Objects are variables, but they contain many values.


              // Object declaration
              var car = {type:"Fiat", model:"500", color:"white"};
              console.log(car) // outputs { type: 'Fiat', model: '500', color: 'white' }

Values inside objects are written in the form of name: value pairs

To access properties of objects, you can use of two ways: 


              objectName.propertyName // This is called 'dot notation'
              // or
              objectName.["propertyName"] // This is called 'bracket notation'

In this context we can use


              console.log(car.type) // prints Fiat
              console.log(car.model) // prints 500
              console.log(car["color"]) // prints white

There are multiple ways of creating Objects. Let's see some of them

Object Creation using 'Object' keyword

A JavaScript object can also be declared using 'Object' keyword. By creating objects this way, it is easier to create properties on the fly.


              const myProfessor = new Object()
              myProfessor['name'] = 'Mark Williams'
              myProfessor['profession'] = 'Professor'
              myProfessor['salary'] = 10
              console.log(myProfessor) // prints { name: 'Mark Williams', profession: 'Professor', salary: 10 }

Object creation using a constructor

In the above examples we've created single objects with some properties, but what if we have to create many objects. To create multiple objects of the same 'type' we can create objects using constructors


              // Creating objects with constructor
              function Car(type, model, color){
              this.type = type
              this.model = model
              this.color = color
              }

              const car_constructor = new Car("Hyundai", "600", "Red")
              console.log(car_constructor) // Car { type: 'Hyundai', model: '600', color: 'Red' }

Objects are created by calling the constructor function by using 'new' keyword

Object creation using a Class

JavaScript objects can also be created using a 'Class'. Classes are special types of functions


              // Creating objects with a class
              class Car_Class {
                  constructor(type, model, color){
                  this.type = type
                  this.model = model
                  this.color = color
                  }
              }

              const car_cls = new Car_Class("Hyundai", "600", "Black")
              console.log(car_cls) // Car_Class { type: 'Hyundai', model: '600', color: 'Black' }
