Tutorial over Variables, Operators & Hoisting

Values and Types

Numbers

JavaScript only has one type of number. Numbers can be written in the following ways

 
        var number1 = 5.75; // 5.75 --> A number with decimals
        var number2 = 5; // 5
        var number4 = 1546e6; // 1546000000 --> Numbers in exponential form
        var number5 = 1345e-5; // 0.01345

Different arithmetic operations with Numbers in JavaScript 


        var number1 = 5.75; // Number declaration
        var number2 = 5;

        var jsAdd = number1 + number2; // 10.75 --> Addition
        var jsSub = number1 - number2; // 0.75 --> Subtraction
        var jsMul = number1 * number2; // 28.75 --> Multiplication
        var jsDiv = number1 / number2; // 1.15 --> Division

Strings

A JavaScript String is zero or more characters enclosed in single ' ' or double quotes " "


        var jsString = "Hello World!" // Hello World!

        var jsStringdq = "Hello World!" // Double Quotes
        var jsStringsq = 'Hello World!' // Single Quotes

Unary Operators

Unary Operators take a single operand and performs an operation. Unary operators cannot be overridden so their functionality is guaranteed.

These are a few types of Unary Operators:

typeof

Returns data type of the operand


        var jsString = "Hello World"
        typeof(jsString) // returns 'string'

        var jsNum = 20
        typeof(jsNum) // returns 'number'

        typeof(true) // returns 'boolean'
        typeof(false) // returns 'boolean'

Unary plus (+)

Converts the operand into a number

If the operand cannot be converted into a number returns NaN


        ar a = 3
        +a // returns '3'

        var b = "Hello"
        +b // returns NaN

Unary minus (-)

Converts the operand into a number and negates it

If the operand cannot be converted into a number returns NaN


        ar a = 3
        -a // returns '-3'
                  
        var b = "Hello"
        -b // returns NaN

Logical Not (!)

Converts the operand into it's boolean equivalent and returns it's opposite


        var a = true
        !a // returns 'false'

        var b = false
        !b // returns 'true'

Increment (++)

This operand adds one to its value and returns it


        var aaa = 5
        var bbb = 5
        console.log(++aaa)
        console.log(bbb++)
        console.log(bbb)

Decrement (--)

This operand subtracts one to its value and returns it


        var aaa = 5
        var bbb = 5
        console.log(--aaa)
        console.log(bbb--)
        console.log(bbb)

delete

Deletes values of specific index of an array or a specific property of an object

Return Values

  • true - if successfully deleted the property or if the property does not exist
  • false - if failed to delete an item 

        var numArray = [1,2,3]
        delete numArray[1] // returns 'true'

        var delVar = 10
        delete delVar // returns 'false'

Hoisting

In JavaScript a variable can be used before it is declared.

Hoisting is a default JavaScript behavior to move all the variable declarations to the top.
